## ğŸ¯ Overview

**UrbanMart** is a mid-sized retail chain operating in a metropolitan city. This project provides an interactive web-based dashboard that helps the management team understand:

- Which product categories are performing well
- How sales vary across stores and days
- Which customers are most valuable to the business

The dashboard replaces traditional Excel reports with a modern, interactive web application that can be accessed through any browser.
